NOSQL Webapp
============

 Simple webapp for delivery driver and order management

How to run
==========

 Clone this repo using "git clone" or download as zip file.
 Then using docker compose run the project using "docker-compose up -d".
 Install "neo4j" Python library for neo4j migration script using "pip3 install neo4j".
 Populate neo4j with data by running migrate.py using "python3 migrate.py".
