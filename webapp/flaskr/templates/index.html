Welcome to the School Seminar Work Project!

This project is a web application built using Python Flask and utilizes several different databases, including Redis, MongoDB, and Neo4j.
The main focus of the project is to create a logistic app that helps find the optimal path for orders with drivers.

Redis is used for storing user sessions, which allows the application to maintain state and track user activity as they navigate through the app.

MongoDB is used for storing user, driver, and order information. This allows the application to access and manipulate this data as needed in order to fulfill orders and manage the logistics of the system.

Neo4j is used to find the best path for orders by taking into account various factors such as distance, driver availability, and any constraints or preferences specified by the user. This helps to ensure that orders are routed efficiently and delivered in a timely manner.
